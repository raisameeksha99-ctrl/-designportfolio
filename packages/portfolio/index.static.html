How I work

Design process

A human-centred, iterative approach — looping back as needed.

  1. 01

    Empathize

    Stakeholder interviews, user research, empathy maps. I don't assume — I go find out.

  2. 02

    Define

    Affinity diagrams, personas, journey maps, "How Might We" statements. Frame the right problem before solving it.

  3. 03

    Ideate

    Crazy 8s, user flows, IA diagrams. Generate wide, then narrow through constraints.

  4. 04

    Prototype

    Lo-fi wireframes → hi-fi mockups → interactive Figma prototypes. Make it tangible before investing in build.

  5. 05

    Test & Iterate

    Usability testing with 5–8 participants, documented findings, prioritised fixes. Design is never done in one pass.
